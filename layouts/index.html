What's in the box?

MQTT Broker

The MQTT Broker is the heart of the platform. It ensures delivery of all messages to the connected clients. It also makes sure that, based on your configuration, only authorized clients are allowed to send and receive messages.

RESTful API

The API is used to administrate the platform configuration and all relevant entities. You can natively call the API or use it via one of our SDKs.

Integration

RealMQ integrates seamlessly with any application stack. Use your own identifiers on entities if you like.

Multi-Tenancy

RealMQ comes with multi-tenancy out of the box. This enables you to run several projects in parallel without them interfering each other.

Message Persistence

RealMQ allows you store messages for a configurable period of time or forever. Simply pull the message history from the server to retrieve everything that happened in the past.

Cloud or on-premise

Try out RealMQ and Sign up for free on our real-time cloud. Or get in touch to discuss an on-premise deployment within your data center.

Open Source

RealMQ is open source. If you want to help to improve, extend, and grow the platform you're more than welcome to join us over at GitHub. There's also a Spectrum Community.

Roadmap

We have a public road map with features that we want to implement next. Although we have put a lot of thought into it, you can help us prioritize what comes next.
